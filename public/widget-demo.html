WebAsk widget — local test page

This page is for smoke-testing the widget against a local Next.js dev server. Open it via http://localhost:3000/widget-demo.html (not file://) so the widget script can talk to the API on the same origin.

Before this works, you need to:

  1. Run npm run register -- --domain localhost --origin http://localhost:3000 to create a site_key for local testing.
  2. Run npm run crawl -- <site_key> to populate pages. (For a local-only quick test, you can also crawl any other site you own.)
  3. Replace YOUR_SITE_KEY below with the one printed by the register command.

The widget button should appear in the bottom-right corner of this page.
